### Q: 用lemon做评测机而不是Arbiter？

众所周知，Arbiter的界面非常美观，安全性能很高，成为NOI比赛第一选择。但是由于Arbiter只安装在NOI Linux中，本系统只能使用同样受欢迎的Lemon评测软件。预计LZH Linux2.0会安装我们自己开发的评测软件nomel
